Here are the instructions on how to run our program!

1. Compile *.java

2. run the following command: java PSO NEIGHBORHOOD_TOPOLOGY INCLUDE_SELF INFLUENCE_STRUCTURE SWARM_SIZE NUM_ITER FUNCTION DIMENSIONS
(if you mess up, the program will output the correct arguments you should enter)

The program will output the global best value at each iteration, separated by commas, followed by the mean and the standard deviation.
